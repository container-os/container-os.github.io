# hello yak

Image主的解决方案没有乡异性问题,
然而这类方案通常并没有配置的弹性.
如何操作该应用也必须透过阅读README后才能理解.

yak希望可以提供简化使用者在使用这类方案的难度,
并仍然享有可调配的弹性.

```
 $ yak install hello-world
```
